# 💳 Mini Banking System with Basic AI Analysis

## 📌 Project Description

The Mini Banking System is a simple Python-based application that allows users to perform basic banking operations such as depositing money, withdrawing money, checking account balance, and viewing a mini statement of recent transactions.

In addition to these core features, the project also includes a basic AI-based component that analyzes past transactions and provides simple insights into user spending patterns.

This project is designed to demonstrate how programming and basic AI concepts can be applied to solve real-world problems in a simple and effective way.

---

## 🚀 Features

* Deposit money into account
* Withdraw money with balance validation
* Check current account balance
* View mini statement (last few transactions)
* Data stored using JSON file
* AI-based spending insight (average spending & prediction)

---

## 🛠️ Technologies Used

The project is developed using Python as the main programming language. JSON is used for storing user data such as balance and transaction history. Basic data analysis techniques are used to implement the AI feature for spending insights.

---

## 📁 Project Structure

The project consists of multiple files including main.py, which controls the overall flow of the program, banking.py for handling banking operations, ml_model.py for the AI feature, and data.json for storing user data. The README.md file provides complete information about the project.

---

## ▶️ How to Run the Project

To run the project, first ensure that Python is installed on your system. Then download or clone the project repository and open it in any Python-supported environment such as VS Code or PyCharm. Finally, run the main.py file to start the program and interact with the system.

---

## 🧠 Working of the System

The system starts by loading existing data from a JSON file. It then displays a menu from which the user can choose different operations. Based on the selected option, the system performs the required task such as deposit or withdrawal. After each operation, the data is updated and saved. The AI module analyzes transaction data to provide insights.

---

## 🤖 AI Feature

The AI component of the system analyzes past withdrawal transactions to calculate the average spending. Based on this data, it predicts future expenses and provides simple suggestions to the user regarding their spending habits.

---

## 🎯 Applications

This project can be used for basic personal finance tracking, especially for students. It also serves as a beginner-friendly project for understanding banking systems and learning how AI can be integrated into real-world applications.

---

## ⚠️ Limitations

The system does not include advanced features such as user authentication or database integration. It relies on file storage, and the AI model used is simple and limited in functionality.

---

## 🔮 Future Enhancements

The project can be improved by adding a graphical user interface, implementing a login system, using a proper database, enhancing the AI model, and converting the system into a web or mobile application.
